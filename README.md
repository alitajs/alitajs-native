# @alitajs/native

[![NPM downloads](http://img.shields.io/npm/dm/@alitajs/native.svg?style=flat)](https://npmjs.org/package/@alitajs/native)



## Install

```bash
# or yarn
$ npm install
```

```bash
$ npm run build --watch
$ npm run start
```

## Usage

Configure in `.umirc.js`,

```js
export default {
  plugins: [
    ['@alitajs/native'],
  ],
}
```

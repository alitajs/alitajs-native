# 插件

官方插件: https://capacitorjs.com/docs/apis
社区插件: https://github.com/capacitor-community

```js
[
  '@capacitor/action-sheet',
  '@capacitor/app',
  '@capacitor/app-launcher',
  '@capacitor/browser',
  '@capacitor/camera',
  '@capacitor/clipboard',

  '@capacitor/share',
  '@capacitor/status-bar',

  '@capacitor/toast',

```

## 注意事项

自行根据每个插件文档要求，修改项目中的权限配置等。比如 `camera` 插件要求在项目中添加相机及相册相关使用描述。
